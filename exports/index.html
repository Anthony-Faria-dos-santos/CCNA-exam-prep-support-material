🚨 Problème de rendu Quarto

Le rendu automatique des formats HTML et PDF a échoué en raison de problèmes systémiques avec l'installation Quarto :

  • HTML : Erreur de compilation SASS persistante
  • PDF : Erreur de fichier temporaire lors du traitement des labs

Plusieurs tentatives de résolution ont été effectuées sans succès.

📁 Structure du projet

Le contenu source est disponible dans :

  • modules/ - Modules théoriques
  • labs/ - Laboratoires Packet Tracer
  • quizzes/ - Évaluations et examens blancs
  • references/ - Glossaire et documentation

🛠️ Rendu manuel possible

Pour générer les formats de sortie :

  1. Vérifier l'installation : quarto check
  2. Nettoyer le cache : rm -rf .quarto
  3. Réessayer : quarto render --to html puis quarto render --to pdf
