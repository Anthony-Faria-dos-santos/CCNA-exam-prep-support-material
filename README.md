# CCNA Exam Prep Support Material

> Personal CCNA 200-301 revision guide with structured notes, CLI configuration examples, hands-on labs and exam cheatsheets.

---

## Objectives

- Consolidate core CCNA 200-301 concepts in a concise, practical format
- Provide ready-to-use IOS configuration examples for common scenarios
- Offer hands-on labs to practice troubleshooting and exam-style tasks
- Serve as a last-minute revision reference before the exam

---

## Repository Structure

```
CCNA-exam-prep-support-material/
├── notes/               # Theoretical summaries by CCNA domain
├── configs/             # IOS configuration examples (routing, VLANs, ACLs, NAT...)
├── labs/                # Lab scenarios for Packet Tracer / GNS3 with solutions
├── cheatsheets/         # Quick reference sheets for commands and key concepts
├── quizzes/             # Practice questions in exam format
└── SKILL.md             # AI skill definition for automated guide generation
```

---

## How to Use This Repo

1. Start with `notes/` for the topic you are currently studying
2. Review the associated `configs/` to see the commands in context
3. Run the corresponding lab in `labs/` and try to solve it before checking the solution
4. Use `cheatsheets/` for last-minute revision before the exam
5. Test yourself with `quizzes/` to simulate real exam conditions

---

## Tools & Requirements

- **Cisco Packet Tracer** or **GNS3** / **EVE-NG** for labs
- Basic familiarity with Cisco IOS CLI
- An up-to-date CCNA 200-301 training resource (book, course or videos)
